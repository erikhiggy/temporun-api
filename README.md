# Temporun-api
API for the TempoRun application utilizing Spotify's Web API

## Project Setup

Install dependencies

```npm install```

Set environment variables

```
CLIENT_ID: XXXX
CLIENT_SECRET: XXXX
REDIRECT_URI: XXXX
PORT: XXXX
```

Run code

```npm start```

Access the API via your port or 8888

```App listening on port: ${PORT}```
